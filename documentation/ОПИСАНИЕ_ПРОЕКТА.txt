СТАКЕДИТ КЛОН - ОПИСАНИЕ ПРОЕКТА
=====================================

ОБЩАЯ ИНФОРМАЦИЯ
-----------------
Название: StackEdit Clone
Версия: 0.0.0
Тип: Веб-приложение для редактирования Markdown
Платформа: Браузер (клиентское приложение)

ТЕХНОЛОГИЧЕСКИЙ СТЕК
--------------------
• Frontend Framework: React 19.1.0
• Сборщик: Vite 6.3.5
• Стилизация: Tailwind CSS 4.1.7
• UI Компоненты: Radix UI + shadcn/ui
• Маршрутизация: React Router DOM 7.6.1
• Markdown парсинг: marked 16.3.0, react-markdown 10.1.0
• Синтаксис подсветка: react-syntax-highlighter 15.6.6
• Уведомления: Sonner 2.0.3
• Анимации: Framer Motion 12.15.0
• Экспорт документов: html-to-docx 1.8.0, docx-preview 0.3.4, xlsx 0.18.5
• Менеджер пакетов: pnpm 10.4.1

ОСНОВНОЙ ФУНКЦИОНАЛ
-------------------
1. РЕДАКТИРОВАНИЕ MARKDOWN
   • Редактор с поддержкой Markdown синтаксиса
   • Панель инструментов для быстрого форматирования
   • Поддержка горячих клавиш и автодополнения
   • Система отмены/повтора (Undo/Redo)

2. ПРЕДПРОСМОТР ДОКУМЕНТОВ
   • HTML предпросмотр с подсветкой синтаксиса
   • Word-превью с рендерингом через docx-preview
   • Word Online интеграция для просмотра .docx файлов
   • Переключение между режимами предпросмотра

3. УПРАВЛЕНИЕ ФАЙЛАМИ
   • Создание, переименование, удаление файлов
   • Переключение между файлами через боковую панель
   • Автосохранение в localStorage браузера
   • Drag & Drop для открытия файлов
   • Поддержка кодировок UTF-8 и Windows-1251

4. ЭКСПОРТ ДОКУМЕНТОВ
   • Сохранение в Word (.doc/.docx)
   • Экспорт в Excel (.xlsx)
   • Копирование в буфер обмена (HTML + текст)
   • Поддержка различных форматов экспорта

5. ИНТЕРФЕЙС ПОЛЬЗОВАТЕЛЯ
   • Адаптивный дизайн с возможностью скрытия панелей
   • Современный UI на базе Radix UI компонентов
   • Темная/светлая тема (поддержка next-themes)
   • Интуитивная навигация и управление

АРХИТЕКТУРА ПРОЕКТА
-------------------
src/
├── App.jsx              # Основной компонент приложения
├── components/ui/       # UI компоненты (shadcn/ui)
├── hooks/              # Пользовательские хуки
├── lib/                # Утилиты и вспомогательные функции
└── assets/             # Статические ресурсы

docs/                   # Собранная версия для деплоя
node_modules/           # Зависимости проекта
public/                 # Публичные файлы

КЛЮЧЕВЫЕ ОСОБЕННОСТИ
--------------------
• ЛОКАЛЬНОЕ ХРАНЕНИЕ: Все файлы сохраняются в localStorage браузера
• ОФЛАЙН РАБОТА: Полная функциональность без интернета
• МНОЖЕСТВЕННЫЕ ФОРМАТЫ: Поддержка .md, .txt файлов
• РЕАЛЬНОЕ ВРЕМЯ: Мгновенный предпросмотр изменений
• ПРОИЗВОДИТЕЛЬНОСТЬ: Оптимизированный рендеринг с дебаунсингом
• СОВМЕСТИМОСТЬ: Работает во всех современных браузерах

ТЕХНИЧЕСКИЕ ДЕТАЛИ
------------------
• Сборка: Vite с настройкой base: '/StackEdit/'
• Выходная папка: docs/ (для GitHub Pages)
• Алиасы: @/ для импорта из src/
• Линтинг: ESLint с правилами для React
• TypeScript: Поддержка типов для React

ЗАВИСИМОСТИ
-----------
Основные библиотеки:
- React 19 + React DOM
- Vite + @vitejs/plugin-react
- Tailwind CSS 4 + @tailwindcss/vite
- Radix UI компоненты (полный набор)
- Lucide React иконки
- Markdown обработка (marked, react-markdown)
- Экспорт документов (html-to-docx, xlsx, docx-preview)

Разработка:
- ESLint + плагины для React
- TypeScript типы
- tw-animate-css для анимаций

СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ
---------------------
1. Создание документации в Markdown
2. Быстрое прототипирование текстов
3. Конвертация между форматами документов
4. Офлайн редактирование файлов
5. Совместная работа с документами (через экспорт)

ОГРАНИЧЕНИЯ
-----------
• Нет облачной синхронизации
• Нет системы версий файлов
• Нет совместного редактирования в реальном времени
• Зависимость от localStorage браузера
• Ограниченная поддержка изображений

ПЛАНЫ РАЗВИТИЯ
--------------

СТАТУС ПРОЕКТА
--------------
Активная разработка. Стабильная версия с основным функционалом.
